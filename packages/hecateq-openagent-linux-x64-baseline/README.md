# @hecateq/hecateq-openagent-linux-x64-baseline

Platform-specific binary for `@hecateq/hecateq-openagent` — Linux x64 (glibc, no AVX2 baseline).

This package is installed automatically as an optional dependency of `@hecateq/hecateq-openagent`. It serves as a fallback on CPUs without AVX2 support. You should not install it directly.
